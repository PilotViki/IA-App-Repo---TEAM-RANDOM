# IA - TSP si NLP - Team RANDOM

## Descriere
Aplicatie complexa de Inteligență Artificială cu interfata grafica Tkinter.

## Functionalitati

### TSP (Travelling Salesman Problem)
- Nearest Neighbor (NN)
- Hill Climbing (HC)
- Simulated Annealing (SA)
- Genetic Algorithm (GA)
- Branch & Bound (BKT)

### NLP (Natural Language Processing)
- 7 clasificatori diferiti
- 3 dataset-uri disponibile
- Vectorizare TF-IDF
- Validare incrucisata

## Instalare

```bash
python -m venv ai_env
source ai_env/bin/activate  # Linux/Mac
# sau
ai_env\Scripts\activate     # Windows

pip install -r requirements.txt
python app_ai.py
